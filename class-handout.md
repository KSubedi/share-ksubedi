# Build Your First AI Voice Agent
## 45-minute hands-on workshop

---

### Course Description

AI voice agents are transforming how businesses handle inbound calls, but the landscape is fragmented: dozens of platforms, competing architectures, and no clear starting point. This workshop cuts through the noise. You'll learn the universal principles behind every voice agent platform -- instruction design, knowledge grounding, call routing, tool integration, analytics, and compliance -- then apply them immediately by building a working agent. The hands-on portion uses echowin (no-code, plain-English training, free to start), but every concept, framework, and best practice transfers directly to Vapi, Retell, Synthflow, Bland, or any other platform you choose. Walk out with a live agent you can call, plus a mental model that works everywhere.

---

### Learning Objectives

- Understand the universal voice agent architecture: instructions, knowledgebase, tools, routing, and channels
- Design effective agent instructions using the Greeting → Disambiguation → Resolution → Close framework (platform-agnostic)
- Build a working voice agent from scratch and deploy it to a phone number
- Configure call routing, screening, and human handoff patterns that apply across platforms
- Read voice agent analytics: which metrics matter, what targets to aim for, and how to close the improvement loop
- Evaluate white-label and agency business models for any voice platform
- Recognize voice agent limitations and apply platform-agnostic workarounds

---

### Target Audience

Technical founders, agency owners, solutions engineers, and product builders. No prior voice AI experience required. Bring a laptop and a phone.

### What You'll Build

A live voice agent that:
- Answers inbound calls with a custom persona and structured call flow
- Pulls answers from a real knowledgebase (website + documents + FAQ)
- Routes calls based on caller intent
- Sends follow-ups and books appointments via integrated tools
- Is ready for a live test call by the end of the session

*(Built on echowin during the workshop; the architecture, instruction patterns, and integration logic apply to any platform.)*
